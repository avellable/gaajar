## gaajar
#### Feed the rabbit.
****

**What is this?**

```
gaajar (गाजर) : Carrot; a tapering orange-coloured root eaten as a vegetable much loved by rabbits.
```

A plugin that feeds "Carrot!"s to all the queues bind to the exchange "carrot".

**How to run?**

```
$ make run-broker
```

**How to verify?**

```
$ make tests
```
